# Southern Metro Conference Championship Operations Manual

Tier: PRIMARY

## Overview

This manual outlines operational procedures for all Southern Metro Conference championship events. All host sites are responsible for venue readiness, staff coordination, emergency procedures, and fan experience standards.

## Venue Access

All event staff must check in at the operations command table no later than 90 minutes prior to gate opening.

## Weather Delay Procedures

The Event Director has final authority regarding weather delays and venue evacuation procedures.

## Media Operations

Credentialed media members must display visible credentials at all times inside restricted access areas.
